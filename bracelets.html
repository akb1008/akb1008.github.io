Safety Pins Heart Bracelet Directions

For the Safety Pins Heart Bracelet, I found a very helpful Youtube video for this project made by Creative Commons which gave step by step directions that I could use to get the big picture of this project altogether, as well as my source on craftbits.com. For this bracelet I used about 90 safety pins in which I assembled different beads in a certain pattern each pin. I then put each one onto a cord in a way that I could get them to align together to look like a heart pattern.

Ring Bracelet Handpiece

For the Diamond Handpiece, I found the source The Perfect Pear extremely valuable when working on this piece for it not only gave directions but pictures as well. I worked with wire for this piece in which I bent onto chains to attach to a ring so that they all connected together.
